# Installation instructions

- Install CMU MOSI from kaggle
- Download and add the mmsdk library from github and add it to your python env
- Install OpenFace
- Install dependencies

# Usage

- Run the "Data_Preprocessing.ipynb" notebook to extract features from CMU MOSI dataset
- Run the "Projet_multi.ipynb" notebook to train and evaluate the model

- Run predict_on_video.py to predict sentiment on a new video file (by providing the path to the video as an argument)
- Run live_prediction.py to predict sentiment in real-time using webcam feed
